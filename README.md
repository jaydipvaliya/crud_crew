# Mimic

> An AI agent trained on your own chat history that replies the way you would.

This repository was built for **Track 1: Build Your Own Digital Clone** — a hackathon challenge to create a bot that learns how a specific person writes and responds, based on their real chat history, and replies to new messages on their behalf.

This is the root of the **Mimic** repository. The full working project — app code, docs, and setup instructions — lives inside the [`Mimic-AI/`](./Mimic-AI) folder.
